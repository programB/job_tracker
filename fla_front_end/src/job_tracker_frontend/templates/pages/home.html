Welcome to the Job Tracker ™ 😉 front end web UI

This app uses back end's service API to display the information about scraped job offers in two forms:

  • Statistics of collected offers presented as a histogram. The time span and binning period can be adjusted, also the presented data can be filtered by a few criteria.
  • All collected offers presented as a paginated list. Each entry presents some essential information and contains link to the original offer. Offers are sorted by the date they were posted at.

Finally one can check the status of the back end's component services.
